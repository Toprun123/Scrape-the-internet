Quick Start
For a simple Quick Start just add this in you HTML page's head tag before you use the functions - 
<script src="http://localhost/scrape.min.js" charset="utf-8"></script>
And use it as - 
requestDataByClassname({class : 'maincounter-number',
                          address : 'https://www.worldometers.info/coronavirus/'},
                        scrapejs.TEXT)
  .then(data => {
    // The returned value is a JavaScript Array.
    console.log(data);
    document.getElementById('output').innerHTML = data[0]; // Prints the current covid cases
});
this makes an Asynchronous request to https://www.worldometers.info/coronavirus and retrieves the innerText of the elements on the page that use the class maincounter-number (hence: retrieving the coronavirus data in realtime)
Asynchronous requests are useful as they resolve or break without blocking the main thread and making the page unresponsive for the user, but this data cannot be used outside of .then(data => {}) and might make it a bit difficult to use if your page requires the data some where else in the page for this problem you can easily make a synchronous request by suffixing the function with the keyword Sync, for example - 
var data = requestDataByIdSync({id : 'ip',
                address : 'https://www.ip.fish/'},
                scrapejs.HTML);
console.log(data.innerHTML);
Here the function makes a Synchronous request to https://ip.fish/ and retrieves your ip address from the Element with the id ip this request keeps the page loading untill the data is retrieved but the data can be used anywhere in the code after the request is successful.

There are multiple ways to use the ScrapeJs library, the quickest way is to use our cdn and load it into your html page like this -
<script src="http://localhost/scrape.js" charset="utf-8"></script>
Or alternatively download the file from here and use it as -
<script src="./scrape.js" charset="utf-8"></script>
For faster loading you are recomended to use the minified version - 
<script src="http://localhost/scrape.min.js" charset="utf-8"></script>
download from here and use it as -
<script src="./scrape.min.js" charset="utf-8"></script>

This version contains only Asynchronous functions from the library.

This file loads faster as it is shorter. 
<script src="http://localhost/1.0.0/scrape.js" charset="utf-8"></script>
npm install -g scrapejs@1.0.0

<script src="http://localhost/1.0.0/scrape.min.js" charset="utf-8"></script>
 The library ScrapeJs contains eight functions namely -
  • requestDataByClassname(); ruturns a js promise which resolves-
    1. A HTMLCollection if mode is scrapejs.HTML
    2. A js Array if mode is scrapejs.TEXT
    Read more about the function here
  • requestDataById(); returns a js promise which resolves-
    1. A HTML[element-type]Element (eg: HTMLSpanElement) if mode is scrapejs.HTML
    2. A js String if mode is scrapejs.TEXT
    Read more about the function here
  • getPage(); returns a js promise which resolves-
    1. A HTMLDocument - The HTML code of the page requested
    Read more about the function here
  • getData(); returns a js promise which resolves-
    1. A js String - The contents of the page requested - can be other than html
    Read more about the function here
  • requestDataByClassnameSync(); returns -
    1. A HTMLCollection if mode is scrapejs.HTML
    2. A js Array if mode is scrapejs.TEXT
    Returns the data synchronously which makes the page unresponsive until loaded in the main thread - Best to use in seperate async thread
    Read more about the function here
  • requestDataByIdSync(); returns -
    1. A HTML[element-type]Element (eg: HTMLSpanElement) if mode is scrapejs.HTML
    2. A js String if mode is scrapejs.TEXT
    Returns the data synchronously which makes the page unresponsive until loaded in the main thread - Best to use in seperate async thread
    Read more about the function here
  • getPageSync(); returns -
    1. A HTMLDocument - The HTML code of the page requested
    Returns the data synchronously which makes the page unresponsive until loaded in the main thread - Best to use in seperate async thread
    Read more about the function here
  • getDataSync(); returns -
    1. A js String - The contents of the page requested - can be other than html
    Returns the data synchronously which makes the page unresponsive until loaded in the main thread - Best to use in seperate async thread
    Read more about the function here
