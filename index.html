Why use ScrapeJs?

In order to fetch content from any webpage you can easily use the fetch API or jquery's ajax or http requests. Then why use ScrapeJs? The simple reason is unlike other alternatives, ScrapeJs can scrape content from almost any webpage - even those that don't allow it - this also works when the same-origin-policy is enabled and the webpage does not allow scraping. ScrapeJs also allows synchronous scraping in js which is not supported by some of these alternatives. The process is quite simple - No JavaScript library can bypass the same-origin-policy - Then how does ScrapeJs? We use PHP instead! A few lines of PHP code along with JavaScript - to make it available to you - has helped me build this library without getting checked or worse blocked by the same-origin-policy.
